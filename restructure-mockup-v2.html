Full Tab & Sub-Tab Structure

Complete map of every tab and what it contains after the restructure.

🏠 Dashboard
No sub-tabs. Single page: Quick actions bar → 4 KPI cards → Instrument Health donut (estimated status) → 3 action cards (overdue, approvals, drift) → Area compliance bars → Upcoming 7 days.
🔧 Instruments
No sub-tabs. Unchanged — searchable/filterable register → instrument detail → edit → calibrate. New addition: "View Procedure" link on detail page if a document is linked.
📅 Schedule
2 sub-tabs:
Tab 1: Technician Queue (default) — Shared work list. Instruments added by planner or technician with requested date + priority. Instrument drops off when calibrated.
Tab 2: Planner — Shows overdue / due soon / estimated out-of-tolerance instruments. "+ Queue" button adds to technician queue with date and priority.
📋 Calibrations
2 sub-tabs (order flipped from current):
Tab 1: Activity Log (default) — All recent calibration records. Each row has a 📄 PDF button to generate the calibration certificate. PDF disabled for non-approved records.
Tab 2: Pending Approvals — Records awaiting supervisor review. Approve/reject with notes. Badge count on tab.
🔬 Smart Diagnostics NEW
3 sub-tabs (replaces old Reports):
Tab 1: Recommendations (default) — Advisory suggestions: interval changes, tolerance reviews, transmitter replacements. Categorised as Critical / Advisory / Optimisation. Dismiss button per recommendation.
Tab 2: Drift Alerts — Instruments trending toward tolerance limit. Table: tag, drift rate, estimated current error, tolerance, risk level.
Tab 3: Repeat Failures — 2+ consecutive as-found failures. These need investigation, not just recalibration.
📄 Reports
Kept but refocused as an export centre. Tabs: Compliance Report (PDF export) / Calibration History (search by tag, export PDF cert) / Overdue Register (CSV + PDF). The old "Failed Calibrations" and "History by Tag" tables stay here as exportable views. Smart analysis moves to Smart Diagnostics.
📁 Documents NEW
No sub-tabs. Site-wide document library. Upload calibration procedures, manuals, reference standard certs. Type tags (Procedure / Manual / Certificate). Link documents to instruments — shows as "View Procedure" on instrument detail page.
⚙️ Settings
Unchanged. Site / Profile / Password / Team. Future: Billing tab.
🆘 Support
Unchanged. FAQ / Tutorials / Contact.
